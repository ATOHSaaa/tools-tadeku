Chrome 拡張機能

Aozora Style

青空文庫の作品本文を、自分好みの読み心地に整える拡張機能です。縦書き・ゴシック・余白・行間などをスライダーで調整でき、設定は自動保存されます。

Chrome Web Store 公開準備中

プライバシーポリシー

できること

縦書き・横書き 組み方向をワンタップで切り替え。縦書きは組み方向に合わせた行揃え。
フォントと背景 ゴシック / 明朝、白・クリーム・生成り・セピア・ダーク・夜読みから選択。
文字サイズ・行間・字間 スライダーで細かく調整。読みやすさをその場で試せます。
設定の自動保存 調整内容はブラウザ内に保存。次に開いたときも同じ設定が復元されます。

使い方

1
拡張機能をインストール

Chrome Web Store から追加します（現在準備中）。

2
青空文庫の本文を開く

cards/.../files/...html の作品ページでのみ動作します。

3
右下の Ao ボタンから調整

画面右下のボタンでパネルを開き、スライダーを動かして読み心地を整えます。

Ao
作品一覧や作家ページには適用されません。設定データは端末内のみに保存され、外部サーバーへ送信されません。詳細はプライバシーポリシーをご覧ください。
